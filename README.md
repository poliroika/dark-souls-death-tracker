# Dark Souls Death Counter 💀

Прозрачный оверлей для отображения счётчика смертей в играх FromSoftware.

Работает в связке с **[DSDeaths](https://github.com/quidrex/DSDeaths)** — утилитой, которая читает счётчик смертей напрямую из RAM игры.

## Почему это лучше OCR?

- **Надёжность** — читаем данные из памяти игры, не зависим от яркости/языка/разрешения
- **Точность** — считаем именно счётчик смертей, а не появление надписи "YOU DIED"  
- **Простота** — минимум зависимостей, минимум кода

## Поддерживаемые игры

- DARK SOULS: Prepare To Die Edition
- DARK SOULS: REMASTERED
- DARK SOULS II
- DARK SOULS II: Scholar of the First Sin
- DARK SOULS III
- Sekiro: Shadows Die Twice
- Elden Ring (только offline, нужно отключить EAC)

## Установка

### 1. Установить Python-оверлей

```bash
# Установите uv если ещё нет
pip install uv

# Установите зависимости
uv sync
```

## Запуск

Запустите run.bat от имени администратора

## Управление оверлеем

- **Перетаскивание** — зажмите ЛКМ и двигайте
- **Выход** — закройте консоль или нажмите Ctrl+C

## Как это работает

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  Dark Souls │ --> │  DSDeaths    │ --> │  Оверлей    │
│   (игра)    │     │  (читает    │     │  (Python)   │
│             │     │   из RAM)    │     │             │
└─────────────┘     └──────────────┘     └─────────────┘
                           │
                           v
                    DSDeaths.txt
                           │
                           v
                    ┌─────────────┐
                    │    OBS      │
                    │ (Text GDI+) │
                    └─────────────┘
```

## Elden Ring

Elden Ring использует Easy Anti-Cheat. Чтобы DSDeaths работал:

1. Отключите EAC (играйте только offline!)
2. Погуглите "Elden Ring disable EAC" для инструкций

⚠️ **Не играйте онлайн с отключённым EAC — это бан!**
